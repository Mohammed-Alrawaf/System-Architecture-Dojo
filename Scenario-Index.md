# Scenario Index

This page provides a quick overview of all architecture scenarios in this repository.

---

## Scenario 01 — Real-Time Alert Gateway

**Problem:**  
Partners rely on polling, causing inefficiency and delays.

**Focus:**  
- event-driven architecture  
- integration patterns  
- cost vs performance  

**What it demonstrates:**  
Designing scalable integrations while preserving legacy constraints.

---

## Scenario 02 — RPA API Bridge

**Problem:**  
No API available for external system (UI-only access).

**Focus:**  
- asynchronous processing  
- state management  
- fallback (HITL)  

**What it demonstrates:**  
Bridging unstable systems while maintaining API reliability.

---

## Scenario 03 — KYC Modernization

**Problem:**  
Slow, manual onboarding under regulatory constraints.

**Focus:**  
- deterministic vs AI decisioning  
- compliance and auditability  
- phased modernization  

**What it demonstrates:**  
Balancing innovation with regulatory and operational control.
